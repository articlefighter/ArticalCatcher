## ArticleSpider
> 解析掘金、知乎等网站上的文章，将作者的原文章提取出来，转成 Markdown 文件。

### 技术架构
- 前台: React
- 后台: Node

### 目录结构
ArticleSpider

+-- api // domi 模拟数据平台  
|  
+-- nodeui // 后台服务 
|    
+-- webapp // 前端项目 
|    +--  config // 配置文件 
|    +--  src 
|    |    +--  res // 静态资源文件 
|    |    +--  utils // 通用函数 
|    |    +--  views // UI 
|    |    |    +--  ajax // 请求 
|    |    |    +--  components // 封装组件 
|    |    |    +--  styles // 通用样式
